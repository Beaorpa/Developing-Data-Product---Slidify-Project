Introduction

Flowers and trees is a shiny application developed to show the relationship between the growth of different types of plants and two types of nutrients added to the soil to accelerating this growth.We will going to use two files Flowers and Trees

File Description (Flowers)

The flowers file have this variables

  1. Nutrient : Type of Nitrient
  2. Quantity: Nutrient added to the pant (cl)
  3. Petal size : Size flowers petals in (mm)
  4. Weight : Weight of flowers(gr)
  5. Color Intensity: Intensity of the petals color
  6. Durability : Time elapsed until the flower fades (days)

File Description (Trees)

The flowers file have this variables

  1. Nutrient : Type of Nitrient added to the plant
  2. Quantity: Amount of Nutrient added to the pant (cl)
  3. Leaf size : Size of the trees leaves in (mm)
  4. Height : Height of trees(m)

NavBar Menu

You can access all the option through the Navbar located at the top of Aplication. These are:

  1. Home - Where we show a brief introduction about the shiny aplication
  2. Table and Summary
  3. Plot

Home Option

In the Home option we try to show briefly a nice (with flowers) introduction about the shine aplication.

Table and Summary Option

This option allows:

  1. A Reactive file selection. That is, we can choose a Flowers or Tree selection. Also we can choose the number of rows which we want to show in a table.

2.Having made the selection we can see a summary operation againts the file and a table with the file and the number of the rows that we have choosed.

Plot Option

The Plot Option also allows us to choose files and rows but then we can see a histogram with the selection.

Plot Example
